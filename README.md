# UniaraVirtualParser

A gem to parse the Uniara Virtual system.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'uniara_virtual_parser'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install uniara_virtual_parser
